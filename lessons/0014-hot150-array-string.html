#1 模式：track min/max Easy

121. Best Time to Buy and Sell Stock

一次买卖，最大利润。

模式：遍历时维护"到当前为止的最小价格"，每天算"今天卖能赚多少"，取最大。

public int maxProfit(int[] prices) {
    int minPrice = Integer.MAX_VALUE;
    int maxProfit = 0;
    for (int p : prices) {
        if (p < minPrice) minPrice = p;
        else maxProfit = Math.max(maxProfit, p - minPrice);
    }
    return maxProfit;
}

变体：122. Best Time to Buy and Sell Stock II（多次交易 → 累加所有上升段）

#2 模式：Boyer-Moore 投票 Easy

169. Majority Element

找到出现次数 > n/2 的元素。

模式：候选 + 计数。遇到相同 +1，不同 -1，归零时换候选。最后剩下的就是众数。

public int majorityElement(int[] nums) {
    int candidate = 0, count = 0;
    for (int n : nums) {
        if (count == 0) candidate = n;
        count += (n == candidate) ? 1 : -1;
    }
    return candidate;
}

直觉：众数个数超过一半，和其他数一一抵消后一定剩它。

#3 模式：前缀/后缀 Medium

238. Product of Array Except Self

输出每个位置除自身外的乘积，不能用除法。

模式：两遍遍历。第一遍算每个位置左边的乘积，第二遍乘上右边的乘积。

public int[] productExceptSelf(int[] nums) {
    int n = nums.length;
    int[] ans = new int[n];
    ans[0] = 1;
    for (int i = 1; i < n; i++)          // 左边乘积
        ans[i] = ans[i-1] * nums[i-1];
    int right = 1;
    for (int i = n-1; i >= 0; i--) {      // 乘上右边
        ans[i] *= right;
        right *= nums[i];
    }
    return ans;
}

面试常问：follow-up O(1) 额外空间（不计输出数组）→ 就是上面的答案。

#4 模式：反转技巧 Medium

189. Rotate Array

数组右移 k 步。

模式：三步反转。反转整体 → 反转前 k 个 → 反转后 n-k 个。

public void rotate(int[] nums, int k) {
    int n = nums.length;
    k %= n;
    reverse(nums, 0, n-1);
    reverse(nums, 0, k-1);
    reverse(nums, k, n-1);
}
private void reverse(int[] a, int l, int r) {
    while (l < r) {
        int t = a[l]; a[l] = a[r]; a[r] = t;
        l++; r--;
    }
}

为什么不用额外数组：O(1) 空间，面试倾向于问这个解法。

#5 模式：贪心可达性 Medium

55. Jump Game

能不能跳到最后一个位置。

模式：维护"当前能跳到的最远位置"，遍历每个位置，如果到了最远还没到终点就 false。

public boolean canJump(int[] nums) {
    int maxReach = 0;
    for (int i = 0; i <= maxReach; i++) {
        maxReach = Math.max(maxReach, i + nums[i]);
        if (maxReach >= nums.length - 1) return true;
    }
    return false;
}

注意：遍历条件是 i <= maxReach，不是 i < n——遇到 0 堵路时自动停下。
